GrammarAnalysisCache

class GrammarAnalysisCache<V>

A caching mechanism for GLL parsing results to improve performance in grammar analysis.

This class maintains separate caches for different grammar types (Alpha, Beta, Projection, Exclusion) to avoid redundant parsing of identical graphs. Each cache is keyed by the graph's hash value, computed using InputGraph.hash or InputGraph.hashGWithId.
